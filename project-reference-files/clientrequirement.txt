# 🔵 2. MASTER TABLES

## Employees (EIC Reference)

```sql
CREATE TABLE employees (
    id SERIAL PRIMARY KEY,
    full_name VARCHAR(150) NOT NULL,
    initials VARCHAR(10),
    designation VARCHAR(100),
    mobile VARCHAR(20),
    email VARCHAR(150),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

---

## Clients

```sql
CREATE TABLE clients (
    id SERIAL PRIMARY KEY,
    company_name VARCHAR(255) NOT NULL,
    organization_type organization_type_enum NOT NULL,
    address TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
```

# 🔵 3. CORE PROJECT TABLE

```sql
CREATE TABLE projects (
    id SERIAL PRIMARY KEY,
    jan_number INTEGER UNIQUE NOT NULL,  -- Numeric only as client required
    client_id INTEGER REFERENCES clients(id),
    
    work_name TEXT NOT NULL,
    location VARCHAR(255),
    tender_reference VARCHAR(255),
    financial_year VARCHAR(20),

    status project_status_enum NOT NULL DEFAULT 'ONGOING',

    previous_eic_id INTEGER REFERENCES employees(id),
    current_eic_id INTEGER REFERENCES employees(id),

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

# 📊 Dashboard Aggregation Ready

Now we can easily query:

## 1️⃣ Time Extension Due (Monthly)

```sql
SELECT p.jan_number, pe.extension_to
FROM project_extensions pe
JOIN projects p ON p.id = pe.project_id
WHERE pe.extension_to <= CURRENT_DATE + INTERVAL '30 days'
AND pe.reminder_enabled = TRUE;
```

---

## 2️⃣ Total EMD Submitted Organization-wise

```sql
SELECT c.organization_type,
       SUM(pc.emd_amount) AS total_emd
FROM project_compliance pc
JOIN projects p ON p.id = pc.project_id
JOIN clients c ON c.id = p.client_id
GROUP BY c.organization_type;
```
